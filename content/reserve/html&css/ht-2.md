# 深入了解超文本

## 超链接

`<a href="abc.html">`超链接，`href`属性指向目标网页

**属性**：

`属性名="属性值"` 要加上双引号

## 路径

`../../abc.html`父文件夹的父文件夹下的页面

`abc.html`本文件夹下的页面

**Web中的路径要使用`/`（斜线）作为分隔符，与操作系统无关**
